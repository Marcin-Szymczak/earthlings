engine.graphics.image.Atlas.generate
public void generate(Image image)
Generates an image atlas data
The specified image's first row and first column are special. They should be mostly white( 255, 255, 255 ). Every black( 0,0,0 ) point specifies end of current frame Every red( 255,0,0 ) point specifies the center point of current frame There should be only ONE center point for each frame ( duh )
